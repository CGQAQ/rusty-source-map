# rusty-source-map
source-map npm package written in Rust

## bench

### You should have nodejs-lts and rust toolchain(include cargo) pre-installed.
```bash
git clone --recurse-submodules git@github.com:CGQAQ/rusty-source-map.git rusty-source-map
cd rusty-source-map
./bench.sh
```
